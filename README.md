# CoffeeShop Client-Server System

A distributed system simulating a coffee ordering service using client-server architecture.

---

## 🚀 Features
- Client-server communication system
- Order handling and processing
- Timeout-based request management
- Multi-client support

---

## 🛠️ Tech Stack
- C (Client)
- Java (Server)
- TCP/IP Sockets
